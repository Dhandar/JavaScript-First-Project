# Simon Says Game 🎮

The Simon Says Game is a fun memory-based game built using **HTML, CSS, and JavaScript**.
The game generates a sequence of colors and the player must repeat the same sequence correctly to move to the next level.

## 🚀 Features

- Interactive color buttons
- Random color sequence generation
- Level progression system
- Score tracking
- Game restart on wrong sequence
- Smooth visual effects

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## 🎯 How the Game Works

1. Press any key to start the game.
2. The game will flash a color.
3. The player must click the same color.
4. Each level adds a new color to the sequence.
5. If the player clicks the wrong color, the game ends.
